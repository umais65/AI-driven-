# AgriShield AI - Generative AI Agricultural Care

AgriShield AI is a mobile plant disease classifier and RAG-powered chatbot. It assists farmers and home gardeners in diagnosing plant leaf anomalies and provides organic, chemical, and preventive treatment guides.

## Setup Instructions

1. Configure environment variables in `.env` (using `.env.example` as a template).
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the backend API:
   `python main.py`
